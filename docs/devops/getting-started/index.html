SAP Cloud SDK for Continuous Delivery

Deprecation#

The SAP Cloud SDK for Continuous Delivery and its features have been merged into project "Piper". We recommend using the General Purpose Pipeline of project "Piper" instead of the SAP Cloud SDK pipeline. The reasoning for this change, as well as further information on how to adopt the General Purpose Pipeline, are described in this guide.
